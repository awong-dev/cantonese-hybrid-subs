# JyutpingSubRev1
**Jyutping Romanisation rules for personal names — Legend of the Condor Heroes (射鵰英雄傳)**
Derived from the PersonalNameGlossary and corrections applied during this session.

---

## Overview
All personal names are romanised in Jyutping (without tone marks). The source for each name is the Chinese character form.

---

## Jyutping conventions used
Standard Jyutping romanisation (without tone marks) is used throughout. No conversion from another system is required — Jyutping is the primary romanisation.

---

## Dash conventions

### Compound / clan surnames
Two-syllable surnames are joined with a dash, capitalised.
- 完顏 → Jyun-Ngaan
- 歐陽 → Au-Yeung

### Standard names (1-char surname + 2-char given)
Given name syllables joined with a dash, second syllable lowercase.
- 陸冠英 → Luk Gun-jing
- 梅超風 → Mui Ciu-fung
- 丘處機 → Jau Cyu-gei

### Sobriquets and 2-character standalone names
Both syllables joined with a dash, both capitalised.
- 一燈 → Jat-Dang
- 靈智 → Ling-Ji
- 傻姑 → So-Gu

### Mongolian and foreign names
All syllables joined with dashes, all capitalised.
- 鐵木真 → Tit-Muk-Jan
- 窩闊台 → Wo-Fut-Toi
- 札木合 → Zaat-Muk-Hap

### Suffix forms (diminutives)
Dash before suffix, suffix lowercase.
- 靖兒 → Zing-ji
- 蓉兒 → Jung-ji
- 康兒 → Hong-ji
- 靖哥哥 → Zing-gogo

## Special cases

### 成吉思汗
Rendered as **Great Khan** in both Jyutping and Yale — the English title is used in place of romanisation.

### 丘 (Qiu)
Jyutping: **Jau** (corrected from initial draft which incorrectly used *Kau*)
