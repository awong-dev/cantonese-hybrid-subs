# YaleSubRev1
**Yale Romanisation rules for personal names — Legend of the Condor Heroes (射鵰英雄傳)**
Derived from the PersonalNameGlossary and corrections applied during this session.

---

## Overview
All personal names are romanised in Cantonese Yale (without tone marks). The source for each name is the Chinese character form; Jyutping is used as an intermediate reference but Yale is the final output.

---

## Systematic Jyutping → Yale conversions

### Initials
| Jyutping | Yale |
|---|---|
| z- | j- |
| c- | ch- |
| j- | y- |
| gw- | gw- (unchanged) |
| kw- | kw- (unchanged) |

## Dash conventions

### Compound / clan surnames
Two-syllable surnames are joined with a dash, capitalised.
- 完顏 → Yun-Ngaan
- 歐陽 → Au-Yeung

### Standard names (1-char surname + 2-char given)
Given name syllables joined with a dash, second syllable lowercase.
- 陸冠英 → Luk Gun-ying
- 梅超風 → Mui Chiu-fung
- 丘處機 → Yau Chyu-gei

### Sobriquets and 2-character standalone names
Both syllables joined with a dash, both capitalised.
- 一燈 → Yat-Dang
- 靈智 → Ling-Ji
- 傻姑 → So-Gu

### Mongolian and foreign names
All syllables joined with dashes, all capitalised.
- 鐵木真 → Tit-Muk-Yan
- 窩闊台 → Wo-Fut-Toi
- 札木合 → Jaat-Muk-Hap

### Suffix forms (diminutives)
Dash before suffix, suffix lowercase.
- 靖兒 → Jing-yi
- 蓉兒 → Yung-yi
- 康兒 → Hong-yi
- 靖哥哥 → Jing-gogo

---

## Special cases

### 成吉思汗
Rendered as **Great Khan** in both Jyutping and Yale — the English title is used in place of romanisation.

### 丘 (Qiu)
Yale: **Yau** (Cantonese 丘 = yau, not kau)
